# 🚌 Margshree - Desi Ride Sharing

India's first ride-sharing platform with Cars, Buses, and Battery Rickshaws!

## ✨ Features

### 🚗 Find Rides
- Filter by vehicle type (Car, Bus, Battery Rickshaw)
- Filter by departure city
- Filter by price
- Book rides with one click!

### 🚘 Offer a Ride
- Add your own ride to share with others
- Set your own price and seats
- Choose your vehicle type

### 🌟 About Margshree
- Eco-friendly and affordable rides
- Connect with your community
- Perfect for Indian cities!

## 🛠️ Installation

1. Install dependencies:
```bash
pip install -r margshree_requirements.txt
```

2. Run the app:
```bash
streamlit run margshree_app.py
```
